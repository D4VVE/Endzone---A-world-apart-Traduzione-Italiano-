##########################

TRADUZIONE ITALIANA 
PER IL GIOCO 
ENDZONE - A WORLD APART

##########################

La traduzione del gioco è quasi completa, potrebbero mancare alcune frasi ma nel complesso sembra funzionare.

Per tradurre il gioco seguire questi passi:

1- Entrare nella cartella Localizations del gioco generalmente situata su "C:\Program Files (x86)\Steam\steamapps\common\Endzone - A World Apart\Localizations"

2- Rinominare il file originale English.json in English.json.bak e copiare nella cartella il file English.json che scaricate da qui.

3- Avviate il gioco e godetevi la traduzione

E' un progetto amatoriale e la traduzione potrebbe non risultare perfetta, ma potrebbe tornare utile per chi non è molto familiare con l'inglese. Non credo di apportare aggiornamenti in futuro.

Per poter tornare alla lingua originale cancellate il file English.json e ripristinate English.json.bak
